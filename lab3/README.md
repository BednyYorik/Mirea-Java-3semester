ЛАБОРАТОРНАЯ РАБОТА №3
  
НАСЛЕДОВАНИЕ В JAVA
</h4>

                                              
  **Цель работы:** изучить понятие наследования, и научиться реализовывать наследование в Java.

- **Упражнение 1.**
  Создать абстрактный класс, описывающий посуду(Dish). С помощью наследования реализовать различные виды посуды, имеющие свои свойства и методы. Протестировать работу классов.
  
- **Упражнение 2.**
  Создать абстрактный класс, описывающий собак(Dog). С помощью наследования реализовать различные породы собак. Протестировать работу классов.
                                
- **Упражнение 3.**
  Создать абстрактный класс, описывающий мебель. С помощью наследования реализовать различные виды мебели. Также создать класс FurnitureShop, моделирующий магазин мебели. Протестировать работу классов. 

  
---

  В 3 упрежнении не нужно реализовывать магазин мебели, необходимо просто протестировать работу геттеров/сеттеров.
